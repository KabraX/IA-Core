# Introduction to @Docs in Cursor

Cursor's @Docs feature allows you to incorporate technical documentation directly as context during your AI-powered workflow. By activating this symbol, you can access both pre-indexed documentation from Cursor and external documentation you add, significantly improving the accuracy and consistency of generated suggestions for coding and technical support.

## Some helpful extra-@Docs

- WXT
- Lingui
- Jotai
- Vitepress
- React Query (TanStack Query)
